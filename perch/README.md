# Perch — app source

This directory holds the Xcode project for the Perch macOS app. For what Perch is,
how to build it from source, accounts/pricing, and license, see the repository's
top-level [README](../README.md).

Perch began as a fork of
[TheBoredTeam/boring.notch](https://github.com/TheBoredTeam/boring.notch) (GPL-3.0);
the notch shell and much of the surrounding UI derive from it. See
[../NOTICE.md](../NOTICE.md) for attribution and [../LICENSE](../LICENSE) for the
license.
